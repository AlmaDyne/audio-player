Settings

Assigned Keys

/

The play key/button starts playing audio.

The pause key/button pauses playing audio.

The stop key/button stops playing audio.

/

The step forward key/button allows the user to skip the current audio they are interacting with and start playing the next one in the list.

The sound playback state is preserved in this case.

/

If the user presses/clicks the step back key/button at the beginning of playing audio, it will skip to the beginning of the previous audio track. If the user presses/clicks this key/button during audio track, it will jump to the start of that audio track.

The sound playback state is preserved in any of these cases.

/

The fast-forward key/button allows the user to go fast-forward. If the user holds down this key/button, it will start to move through the audio faster than the playback speed in the forward direction.

When the audio reaches the end, it will jump to the next audio.

The sound playback state is preserved in any of these cases.

/

Holding down the fast-rewind key/button allows the user to rewind audio backward. This is helpful when the user wants to go back to a specific moment in the audio.

When the audio reaches the beginning, it will jump to the previous audio.

The sound playback state is preserved in any of these cases.

/

The volume key/button turns the sound on and off. When on, the last non-zero volume value is applied.

+ /

The louder key increases volume of audio.

+ /

The quieter key decreases volume of audio.

The shuffle button is disabled. The playlist plays in clear order.

The shuffle button is enabled. It is used to randomly play an audio from a given list.

The repeat button is disabled. When an audio track has finished playing, the next track is automatically selected and played.

The repeat button is enabled in playlist mode. When the playlist is finished, the player will stop playing and return to its initial appearance.

The repeat button is enabled in track mode. It is used to play the selected audio on repeat.

The configuration key/button is used to change the scheme of the control buttons layout in the audio player.

The color key/button is used to adjust the color style of the audio player.

Smooth playlist style enabled.

Strict playlist style enabled.

The settings key/button allows users to access the audio player settings to customize some control options and see the current playlist with an ordered set of audio tracks even when shuffling is enabled.

The information key/button is used to get acquainted with the keys assigned to the audio player buttons.

The tracklist database key/button is utilized to toggle the display of available tracklists on and off.

The close key/button closes the active section/window.

When the track from the playlist is focused the specified key (re)starts playing that track from the beginning.

Clicking on the track name performs the same action.

When the database tracklist is focused the specified key expands/collapses the tracklist details.

Clicking on the tracklist name performs the same action.

When the track from the playlist is focused the specified key removes that track from it.

The corresponding button works regardless of focus.

When the database tracklist or any of its active internal elements are focused, the specified key will delete the tracks from that tracklist and from the database. Deleting all tracks prompts to delete the tracklist itself.

The corresponding button works regardless of focus.

This operation is irreversible!

+

When the database tracklist or any of its active internal elements are focused, the specified key combination will clear the playlist and add the selected tracks from that tracklist to the player's playlist.

The corresponding button works regardless of focus.

+

When the database tracklist or any of its active internal elements are focused, the specified key combination will add the selected tracks from that tracklist to the player's playlist without clearing it.

The corresponding button works regardless of focus.

When the database tracklist or any of its active internal elements are focused, the specified key launches the tracklist manager, allowing for editing of the current tracklist, including: updating the tracklist name and image, altering the artist's name, editing the track title and/or file, as well as adding new tracks, removing existing ones, and deleting the tracklist itself.

The corresponding button works regardless of focus.

When the database tracklist or any of its active internal elements are not focused, the specified key allows for the creation of a new tracklist by adding tracks with all the necessary data.

The corresponding button works regardless of focus.

The sort key/button, when date sorting is activated, arranges the tracklists by the date of creation/modification, starting with the most recently created/modified.

The sort key/button, when alphabetical sorting is activated, organizes the tracklists in alphabetical order from A to Z.

The expand details key/button is designed to unveil the full details of each tracklist.

The collapse details key/button is crafted to hide the extended information of each tracklist, ensuring a tidy and concise overview.

The clear playlist key/button swiftly removes all tracks from the current playlist, resetting it to a blank state for new selections.
